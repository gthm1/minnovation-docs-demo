# XVision Installation Guide

XVision is Minnovation's edge vision hardware line. This guide covers unboxing,
mounting, and first-time network setup.

## What's in the box

- XVision camera unit
- Mounting bracket and screws
- Power-over-Ethernet (PoE) injector
- Quick-start card

## Mounting

Mount the unit 2.5–3m above ground level with a clear, unobstructed view of the
monitored area. Avoid direct backlighting where possible.

## Network setup

1. Connect the unit to your network switch using the supplied PoE injector.
2. Open the XVision companion app and select **Add Device**.
3. The app will discover the unit on the local network — follow the prompts to assign
   it to a site and confirm firmware is up to date.
