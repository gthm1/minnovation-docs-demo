# Minnovation Documentation

Welcome to the public documentation hub for Minnovation Technologies' platforms.

This is a **demo site** built to evaluate GitHub + Docusaurus + a git-based CMS as an
option for separating public, client-facing documentation from internal-only content.

## What's in this demo

- **Public Docs** (this section) — client-facing guides for AlphaX Cloud, the Conduit
  API, and XVision, mirroring what currently lives in GitBook and HubSpot.
- **Internal Docs** — a separate content instance for engineering/support-only material,
  built to show how a second, structurally distinct space would work.
- **/admin** — a lightweight CMS (Decap CMS) so non-technical writers can edit pages
  through a form-based UI instead of raw Markdown or git commands.

Use the navigation bar above to switch between **Public Docs** and **Internal Docs**.
